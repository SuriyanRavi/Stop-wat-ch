# ⏱️ Stop Timer

A clean and responsive **Stopwatch / Stop Timer** web application built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## 📸 Preview

> Open `index.html` in any modern browser to see it in action.

---

## ✨ Features

- ▶️ **Start** — Begin counting time from zero (or resume from a paused state)
- ⏸️ **Pause** — Pause the timer at any point
- 🔄 **Reset** — Reset the timer back to `00:00:00.000`
- 🎯 **Lap Recording** — Record lap times and view them in a list
- 📱 **Responsive Design** — Works seamlessly on desktop and mobile devices
- ⚡ **Lightweight** — Pure HTML, CSS, and JavaScript — zero dependencies

## 🚀 Getting Started

### Prerequisites

No installations required! Just a modern web browser (Chrome, Firefox, Edge, Safari).

### Running the App

1. **Clone or download** this repository:
   ```bash
   git clone https://github.com/your-username/Stop-Timmer.git
   ```

2. **Navigate** into the project folder:
   ```bash
   cd Stop-Timmer
   ```

3. **Open** `index.html` in your browser:
   - Double-click `index.html`, or
   - Right-click → *Open with* → your preferred browser

## 📖 How to Use

1. Click **Start** to begin the stopwatch.
2. Click **Pause** to freeze the current time.
3. Click **Start** again to resume from where you left off.
4. Click **Lap** (if available) to record the current time as a lap.
5. Click **Reset** to clear everything and return to `00:00:00`.
